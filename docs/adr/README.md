# Architecture Decision Records

Crie um ADR quando uma decisão técnica for duradoura, tiver alternativas relevantes ou
for difícil de reverter. Use `NNNN-titulo-curto.md` e mantenha o documento conciso:

```md
# ADR NNNN — Título

**Status:** Proposto | Aceito | Substituído
**Data:** AAAA-MM-DD

## Contexto

## Decisão

## Alternativas consideradas

## Consequências
```
